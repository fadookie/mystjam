ArialDni v.1.0

TO USE IN MICROSOFT WORD...

1) Click Format
2) Select Font...
3) Click Character Spacing tab
4) Click the "Kerning for fonts" checkbox
5) Set "Points and above" to 2 (you'll need to type it in)
6) Click OK

And you're all set!

As far as I know, the font works just fine in other applications... if not, please let me know.  Of course, feedback, comments, questions, etc. are welcome!  Please drop me a note at dnigrammar@yahoo.com to let me know what you think.
